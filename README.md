## Requirements

- Node.js: Make sure you have Node.js installed on your machine.

## Installation

1. Clone this repository:

```bash
git clone https://github.com/joaomichalskiTI/carrinho_de_compras.git
```

## Install dependencies:

```bash
npm
```

## Start Development:

```bash
npm run dev
```

## Start Prodution:

```bash
npm run build --> npm run start
```
